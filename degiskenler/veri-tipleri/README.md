# Veri Tipleri
Java'da veri tipleri, bir değişkenin hangi türde bilgi tutacağını belirler.

Veri tipleri ikiye ayrılır:
1. Primitive (ilkel) veri tipleri
2. Reference (referans) veri tipleri

Primitive veri tipleri doğrudan değerin kendisini tutar.
- int    → tam sayılar
- double → ondalıklı sayılar
- boolean → true / false
- char   → tek karakter '' tek tırnakla tutar.
  
Reference veri tipleri, değerin kendisini değil referansını tutar.
En yaygın örnek:
- String
